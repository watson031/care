Listes des differents status code

Pour les requetes de type GET

  • 200 : Tout va bien
  • 404 : si aucun objet n'est retourné (ex: id non valide)

Pour les requetes de type POST et PUT

  • 201 : si ajouté ou modifié
  • 505 : si pas possible de modifié
  • 444 : si email existant
  • 455 : si code acces invalide

Pour les requetes de type DELETE / DESACTIVATE

  • 222 : si supprimé ou desactivé
  • 409 : s'il y'a eu un probleme pour supprimer (ex: id non trouvé)

Pour les envois des objets avec images

  • 223 : si tout s'est bien passé
  • 432 : si mauvais extension
  • 433 : si depasse la taille maximum (<1 mo)
